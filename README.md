# GG Demon (Battle Charge)

## 概要 (Overview)
「Battle Charge」は、エネルギー管理と読み合いが鍵となるターン制戦略バトルゲームです。

## タワーモード (Tower Mode)
HPを引き継いで塔を登るサバイバルモード。

### 特徴
*   **ダブル・デッキシステム**: 敵キャラクターとボスキャラクターはそれぞれ「デッキ」から選出され、一定周期（ザコ9回、ボス5回）ですべての種類が出現するよう制御されています。これにより、「同じ敵ばかり出る」偏りを防ぎます。
*   **難易度スケーリング**: 階層深さに応じて、敵に付与される特性（接頭辞）の数やAIの賢さ、パッシブスキルの種類が変化し、徐々に難易度が上がります。
*   **アイテム**: 宝箱から獲得できる強化効果（メリット）と弱体化効果（デメリット）のセット。
*   **新スキル**: ミミック（宝箱モンスター）を倒すと獲得できる強力な行動。
*   **25階層ループ**: ボスは5階層ごとに出現し、25階層ですべてのボスを倒破するサイクルとなります。
