# G.G Demon Master Edition

G.G Demonは、高度な戦略と心理戦が楽しめるHTML5ベースのカードバトルゲームです。
1vs1の対CPU戦、タワー攻略モード、そしてオンライン対戦に対応しています。

## 概要

プレイヤーは様々な能力を持つ「ヒーロー」を選択し、相手のHPを0にするか、勝利に必要なエネルギー(WinE)を溜めることを目指します。
ターン制で進行し、互いにコマンド（CHARGE, ATTACK, GUARD, SKILL）を同時に公開して勝敗を決します。

## 開始方法 (Usage)

このゲームはローカルサーバー上で動作します。以下のいずれかの方法で起動してください。

### Windows / PowerShell ユーザー
フォルダ内の `start-server.ps1` を実行してください。自動的にブラウザが立ち上がります。
```powershell
./start-server.ps1
```

### Node.js ユーザー
Node.jsがインストールされている場合、以下のコマンドで起動できます。
```bash
npm install
npm run dev
```

### その他
Python等が使える環境であれば、ルートディレクトリで簡易HTTPサーバーを立ち上げることでプレイ可能です。
```bash
python -m http.server 8001
# ブラウザで http://localhost:8001 にアクセス
```

## 基本ルール

### 勝利条件
以下の**どちらか**を満たすと勝利となります。
1. **KO勝利**: 相手のHPを0にする。
2. **エネルギー勝利**: 自分のエネルギーが、キャラクター固有の「勝利必要エネルギー (WinE)」に達する。

### コマンド
毎ターン、以下の4つから行動を選択します。

| コマンド | アイコン | 説明 |
| :--- | :---: | :--- |
| **CHARGE** | ⚡ | エネルギーを溜めます。基本増加量はキャラクターの `ChgE` (Charge Efficiency) に依存します。 |
| **ATTACK** | ⚔️ | エネルギーを消費(`AtkC`)して攻撃します。相手に `ATK` 分のダメージを与えます。 |
| **GUARD** | 🛡️ | エネルギーを消費(`GrdC`)して防御します。相手のATTACKを無効化します。 |
| **SKILL** | ⭐ | キャラクターやタワーモードで獲得した特殊能力を使用します。 |

### ステータス用語
- **HP**: 体力。0になると敗北。
- **ATK**: 攻撃力。ATTACK成功時のダメージ量。
- **WinE**: 勝利に必要なエネルギー量。
- **ChgE**: CHARGE時のエネルギー増加量。
- **GrdC**: GUARDに必要なコスト。
- **AtkC**: ATTACKに必要なコスト。
- **StartE**: ゲーム開始時の初期エネルギー。

### 敵特性 (Enemy Traits)
通常の敵にはランダムで特性が付与されることがあります。

- **迅雷の**: チャージ速度特化 (ChgE+1, HP-1)
- **鋼鉄の**: 圧倒的な耐久力 (HP+3, AtkC+1)
- **狂乱の**: 暴走型 (ATK+2, WinE+2)
- **狡猾の**: 守りが固い (GrdC-1, WinE-1)
- **神速の**: 短期決戦型 (StartE+2, HP-2)

### アイテム効果 (Item Effects)

**メリット (Merits)**
- **ATK_UP**: ATKアップ
- **HP_UP**: 最大HPアップ
- **CHGE_UP**: チャージ効率アップ
- **WINE_DOWN**: 勝利必要Eダウン
- **GRDC_DOWN**: ガードコストダウン
- **ATKC_DOWN**: 攻撃コストダウン
- **START_E_UP**: 初期エネルギーアップ
- **SKILL_EFFECT_UP**: スキル効果量アップ
- **SKILL_COST_DOWN**: スキルコストダウン
- **CHARGE_COST_DOWN**: チャージコストダウン

**デメリット (Demerits)**
- 通常ステータスの逆効果 (ATK_DOWN, HP_DOWN等)
- **WINE_UP**: 勝利必要Eアップ
- **GRDC_UP**: ガードコストアップ
- **ATKC_UP**: 攻撃コストアップ
- **SKILL_COST_UP**: スキルコストアップ
